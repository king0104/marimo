- Flutter에서 스플래시 스크린을 구현하고, 이를 문서화했습니다.
- 로그인 스크린 UI를 구현했습니다.

<br>

### ❣️ Today I've Learned
- Central: BLE 장치를 스캔하고 연결하는 역할 (예: 스마트폰).
- Peripheral: 데이터를 광고하고 연결 요청을 수락하는 역할 (예: OBD2 장치).
- Service: 특정 기능을 그룹화하는 데이터 구조.
- Characteristic: 데이터를 저장하는 기본 단위로, Read/Write/Notify 속성을 가짐.
- Descriptor: Characteristic의 부가 정보를 포함하는 요소.
